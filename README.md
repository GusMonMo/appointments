Lista de decisões do projeto:
Framework: React;
Bundler: Vite;
Estilização: Tailwindcss + lucide-react;
Componentes prontos: Shadcn;
Formulários + validação: React Hook Form + Zod;
Calendários: Fullcalendar, dayjs, date-fns;

Arquitetura: Monolito;

Fiz uso de agentes de IA para desenvolvimento rápido de código e estilização baseado nas decisões abordadas, o que cooperou para um resultado satisfatório e ágil.
